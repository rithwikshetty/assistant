# Code of Conduct

This repository is intended to be a respectful, professional place to collaborate.

## Expected Behavior

- Be respectful in discussions, reviews, and issue reports.
- Focus feedback on the code, behavior, or proposal, not on the person.
- Assume good intent, but be direct and specific when raising concerns.
- Keep discussions relevant to the project and its maintenance.

## Unacceptable Behavior

- Harassment, abuse, discrimination, or personal attacks.
- Bad-faith disruption, trolling, or repeated antagonistic behavior.
- Sharing private or sensitive information without permission.
- Spam, solicitation, or deliberately misleading technical claims.

## Enforcement

The repository maintainer may remove comments, issues, pull requests, or contributors that violate these standards.

## Reporting

If you experience or observe unacceptable behavior, report it privately to the repository maintainer.
