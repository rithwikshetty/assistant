# Workspace Basics

Use this skill as a lightweight execution checklist.

## Core workflow

1. Restate the goal in one sentence.
2. Identify the source of truth before answering.
3. Prefer uploaded files and project files for workspace-specific questions.
4. Use web search only for current or external facts.
5. Show assumptions clearly when the source of truth is incomplete.

## File-backed answers

- Read the relevant file before summarising it.
- Quote only short passages when necessary.
- If the file does not contain the answer, say that directly instead of guessing.

## Research tasks

- Start broad enough to avoid false negatives.
- Narrow quickly once you have the right source.
- Keep only the evidence that changes the answer.

## Writing tasks

- Match the user’s requested format.
- Keep claims grounded in provided material.
- Prefer direct, plain language over decorative wording.

## Final check

- Confirm every user requirement was addressed.
- Confirm the answer distinguishes facts, assumptions, and recommendations.
